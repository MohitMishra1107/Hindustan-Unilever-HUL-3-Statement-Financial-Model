# Hindustan-Unilever-HUL-3-Statement-Financial-Model
Built an integrated 3-statement financial model (IS, BS, CF) in Excel using FY21–FY25 historicals (₹45,311 Cr → ₹60,680 Cr revenue), with fully linked projections for FY26–FY28 Projected 6–7% revenue growth and stable ~23.5% operating margins, forecasting FY26 revenue of ₹64,321 Cr and sustained profitability trends .
